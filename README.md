# GuidEx - Interactive Authentication Flow

![GuidEx Authentication Flow](./public/preview.png)

An interactive, highly dynamic authentication experience (Login & Signup) built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features
- **Interactive Mascot**: A friendly mascot that reacts to user actions:
  - 👀 Idle floating and blinking.
  - 🤩 Gets excited when you type an email.
  - 😵 Looks confused if you leave a field empty.
  - 😠 Shows an angry face on errors or invalid passwords.
  - 🥰 Shows a happy face for strong passwords.
  - 🎉 Celebrates successful form submissions.
- **Dynamic Background Illusions**: Beautiful, animated background blobs (purple, pink, orange, green) that shift and morph to create an immersive environment. They react alongside the mascot (e.g., turning red on error, celebrating on success).
- **Responsive Layout**: Designed for a seamless, premium experience on both desktop and mobile devices.
- **Modern Tech Stack**: Bootstrapped with Vite, React, TypeScript, and styled with Tailwind CSS.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` to view the app in your browser.

## Technologies Used
- **React** & **TypeScript**
- **Vite**
- **Framer Motion** (for smooth spring physics and animations)
- **Tailwind CSS** (for responsive styling)
- **Lucide React** (for icons)
